# NLP-Job-Prep-App

The Git repository is a PoC that showcases the potential of GPT-3 and prompt engineering in NLP education. By prompting GPT-3 with NLP articles from Wikipedia, the repository generates questions and answers related to NLP concepts, acting as a question-answering system that can help users prepare for exams and job interviews. The repository demonstrates the capabilities of these technologies and offers a glimpse of what's possible in the field of NLP.
